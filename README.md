# Database-Driven Shopping Cart & Order Management System

This project implements a complete mini e-commerce system using PostgreSQL, FastAPI, and vanilla HTML/CSS/JS.

## Structure

```text
project/
│── backend/
│── frontend/
│── database/
│   ├── schema.sql
│   ├── queries.sql
│   ├── procedures.sql
│   ├── triggers.sql
│── README.md
```

Implementation details, setup steps, API docs, and demo flow are added in later checkpoints.
